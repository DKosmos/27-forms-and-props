# Reddit Search

Search for a Subreddit and return the top X articles from 0-100.

## Use

Launch from console with `npm run watch`, then test in browser at `localhost:8080`.

## Dependencies

- babel-core
- babel-loader
- babel-preset-env
- babel-preset-react
- css-loader
- extract-text-webpack-plugin
- html-webpack-plugin
- node-sass
- react
- react-dom
- sass-loader
- superagent
- webpack
- webpack-dev-server
